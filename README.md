> Highest Scores app for Outline

## Install

```sh
npm install
```

## Usage

Use num argument to pass in the number of lines you want to output.

```sh
npm run start -- --num=[n]
# or
node ./highestscores.js --num=[n]
```

## Run tests

```sh
npm run test
```
